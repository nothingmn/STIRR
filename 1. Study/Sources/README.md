# Sources

Source material, references, and bibliographic records — the evidentiary foundation of the Study. This area holds the source material that feeds research and writing: books, papers, articles, conversations, datasets, and any external reference. Sources are the ground truth against which claims are checked.

## What belongs here

- Bibliographic records for books, papers, and articles
- Annotations and marginalia tied to specific sources
- PDFs or transcriptions of key source material (or links to them)
- Conversation logs that serve as primary sources
- Structured reference entries with metadata (author, date, DOI, etc.)

## What does NOT belong here

- Analysis or argument built from sources → [[../Research/README.md|Research]]
- Web clippings without academic or research value → [[../../7. Archive/Clippings/README.md|Clippings]]
- General reference on stable topics → [[../../2. Library/General Reference/README.md|General Reference]]
- Drafts of your own writing → [[../Writing/README.md|Writing]]

## Tips

- **One source, one note.** Each source gets its own file. Do not combine multiple sources into a single entry.
- **Metadata is mandatory.** Every source note must include at minimum: author, title, and date (or "n.d."). Add DOI, URL, or ISBN when available.
- **Link up.** Source notes should be linked from every research or synthesis note that draws on them. Downstream notes depend on these links for provenance.
- **Cite consistently.** Adopt a single citation style across all source notes. Consistency enables automated tooling and reliable cross-referencing.
