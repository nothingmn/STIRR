# General Reference

Topics not yet warranting their own wing — the stable, encyclopedic knowledge base.

## What belongs here

- Encyclopedic entries on topics, concepts, and definitions
- How-to guides and technical references
- Glossaries, terminologies, and cheat sheets
- Stable knowledge unlikely to change frequently
- Topics being indexed for potential future wing creation

## What does NOT belong here

- Active research notes (those belong in [[../../1. Study/Research/PROTOCOL|Research Protocol]])
- Source records tied to specific bibliographic entries (those belong in [[../../1. Study/Sources/PROTOCOL|Sources Protocol]])
- Project-specific reference (that lives with the project in Workshop)
- Obsolete or superseded material (move to [[../../7. Archive/Obsolete/PROTOCOL|Obsolete Protocol]])
- Personal notes (those belong in [[../../5. Quarters/Living/PROTOCOL|Living Protocol]])

## Area-specific rules

1. **Precision indexing.** The LLM actively indexes and cross-references these notes. It may create new reference entries, update existing ones, and build internal links — all subject to human review if flagged.
2. **No orphans.** Every reference note should link to at least one other note — either another reference entry, a research note, or a project note. Isolated notes are flags for re-evaluation.
